# Algorithm Design & Analysis Project

A collection of algorithm implementations covering Divide and Conquer, Dynamic Programming,
Greedy, Brute Force, and Iterative Improvement techniques.
Built as a team project at Ain Shams University.

## My Contribution
- Task 7: Job Scheduling with Penalty Minimization (Dynamic Programming, Brute Force, Divide and Conquer)

## Tasks

| Task | Problem | Techniques |
|------|---------|------------|
| 1 | Tromino Tiling on 2n×2n board with 3-coloring | Divide and Conquer, Brute Force |
| 2 | Knight's Tour on n×n chessboard (closed tour) | Greedy |
| 3 | 8-disk 4-peg Tower of Hanoi (33 moves) | Divide and Conquer, Dynamic Programming |
| 4 | Knight exchange on 3×4 chessboard | Iterative Improvement |
| 5 | Shooter vs. moving target (n hiding spots) | Divide and Conquer |
| 6 | n×n point lattice crossing with 2n−2 lines | Dynamic Programming |
| 7 | Job scheduling — minimize penalty ← my contribution | Dynamic Programming, Brute Force, Divide and Conquer |
| 8 | Approximate minimum cut on weighted graph | Iterative Improvement, Brute Force |
| 9 | K-means clustering | Iterative Improvement, Divide and Conquer, Brute Force |
| 10 | Coin removal puzzle (heads/tails line) | — |

## Deliverables
